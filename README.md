# Employee Management System (Core Java)

A menu-driven console application developed using Core Java to manage employee records.

## Features
- Add employee
- View all employees

## Concepts Used
- Object-Oriented Programming (OOP)
- Encapsulation
- Java Collections (ArrayList)
- Scanner for user input

## How to Run
Compile and run the application:
1.javac Employee.java EmployeeManagementApp.java
2.java EmployeeManagementApp
